# 🛒 RealMart Sales Data Analysis Dashboard

An interactive **Power BI dashboard** built to analyze retail sales performance, customer behavior, and business trends using real-world styled data.

---

## 📊 Project Overview

This project focuses on **data analysis and visualization** of a retail dataset to uncover meaningful insights about sales, orders, returns, and customer preferences.

The dashboard helps in understanding:
- Sales growth over time
- Category-wise performance
- Customer purchase behavior
- Payment trends and return patterns

---

## 🚀 Key Features

- 📈 **Sales Analysis**
  - Total Sales: **4.13M**
  - Year-wise growth trends

- 🧾 **Order Insights**
  - Total Orders: **1378**
  - Total Quantity Sold: **1892**

- 🔄 **Returns Analysis**
  - Identified return trends and patterns

- 🛍️ **Category & Sub-category Analysis**
  - Clothing, Electronics, Furniture performance
  - Sub-category revenue comparison

- 💳 **Payment Mode Distribution**
  - COD, UPI, EMI, Credit/Debit Cards

---

## 🧠 Key Insights

- Sales showed **consistent growth over the years**
- Certain sub-categories contributed significantly to revenue
- COD and UPI were dominant payment methods
- Return rate was relatively low but present in specific segments
- Customer behavior varies across product categories
